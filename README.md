# Market Analytics Platform

Backend platform for collecting market data, monitoring signals and generating analytical insights from multiple sources.

The project combines API integrations, technical indicators, AI-assisted analysis and Telegram-based delivery workflows.

---

## Business Context

Financial and crypto markets generate large amounts of fragmented information:

- exchange data
- market volatility
- news feeds
- sentiment signals
- technical indicators

Operational teams and analysts need centralized tooling for monitoring and structured signal delivery.

---

## Key Capabilities

- Multi-source market data collection
- Technical indicator processing
- AI-assisted analytical workflows
- Telegram notifications
- Signal aggregation
- Risk-oriented architecture
- Backend-oriented modular structure

---

## Tech Stack

- Python 3.11+
- FastAPI
- Telegram Bot API
- Pandas
- OpenAI / DeepSeek integrations
- Exchange API integrations
- Linux VPS deployment
